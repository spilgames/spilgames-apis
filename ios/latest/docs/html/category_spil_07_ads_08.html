- (BOOL) adsRequestInGameAd: (CGSize)  size

Makes a request to get an advert (on the default location) and return it to the invoker when it's done through the InGameAdsDelegate implementation set up prior the call to this method. This methods returns right away and gives NO if there is not a valid chartboost instance, YES otherwise. Although this method returns YES, it doesn't mean the ad will be in fact retrieved and returned.
